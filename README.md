# 🍔 Food Delivery Analytics Dashboard

## 📌 Project Overview

The Food Delivery Analytics Dashboard is a Power BI project developed to analyze food delivery operations and business performance. The dashboard provides insights into revenue, order volume, delivery efficiency, and restaurant performance through interactive visualizations and KPI metrics.

This project demonstrates the complete data analytics workflow, including data cleaning, transformation, DAX calculations, and dashboard development.

---

## 🎯 Project Objectives

* Monitor overall business performance.
* Analyze restaurant-wise revenue generation.
* Track delivery efficiency and delivery times.
* Compare restaurant performance.
* Generate actionable business insights through data visualization.

---

---

## ⚠️ Data Quality Issues Identified

The raw dataset contained several inconsistencies:

* Delivery Time values such as "30mins"
* Amount values containing currency text (e.g., "500Rs")
* Null Order IDs
* N/A values
* Inconsistent restaurant names

---

## 🧹 Data Cleaning & Transformation

Data cleaning was performed using **Power Query**:

* Removed null values
* Corrected data types
* Cleaned Delivery Time column
* Cleaned Amount column
* Standardized Restaurant names
* Removed unwanted text and symbols

---

## 🧮 DAX Measures Implemented

```DAX
Total Revenue =
SUM('Dirty_FoodDelivery_Analytics_900'[Amount])

Order Count =
COUNT('Dirty_FoodDelivery_Analytics_900'[OrderID])

Average Delivery Time =
AVERAGE('Dirty_FoodDelivery_Analytics_900'[DeliveryTime])
```

### Power Query Custom Column

```Power Query
Delivery Status 
= Table.AddColumn(
    #"Trimmed Text",
    "Delivery Status",
    each if [DeliveryTime] > 30
    then "Late"
    else "On Time"
)
```
